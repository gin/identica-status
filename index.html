Identica Status by gin

Widget for Wordpress to display last updates from identi.ca accounts.

Dependencies

Wordpress 3.+

Install

To install manually: Download the latest version of the plugin to the server. Copy the tarball file to the /wp-content/plugins folder and unpack it. In your WordPress administration panel, click on Plugins from the menu at the top. You should see your new plugin listed. To turn the WordPress plugin on, click Activate on the far right side of the list.
